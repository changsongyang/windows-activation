### Step 2  
Install KMS client key.

Use this command:

```CSS
slmgr /ipk yourlicensekey
```

> [!NOTE]
> Please select one of the **license keys** from the list that **matches your version of Windows** and replace it with the phrase `yourlicencekey` in the command.
